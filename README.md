# Stylaxx Compiler üöÄ

**Stylaxx** is a custom programming language built from scratch using **C**, **Flex** (Lexical Analyzer), and **Bison** (Parser Generator). It features a unique syntax inspired by Bengali phonetic words (Banglish) and supports advanced mathematical operations, control flow, loops, and arrays.

## ‚ú® Features

- **Data Types:** Supports Integer, Float, and String.
- **Variables:** Dynamic variable declaration using `dhori`.
- **Input/Output:** Console input (`nio`) and output (`dekhao`).
- **Control Flow:** `If-Else` logic (`jodi`, `nahole`).
- **Loops:** `While` loop (`jotokhon`) and `For` loop (`ghuro`).
- **Arrays:** 1D Array support for data storage.
- **Math Library:** Built-in functions for `pow`, `root`, `sin`, `cos`, `tan`, `abs`.
- **Comments:** Supports single-line comments (`//`).
- **Error Handling:** Detects syntax errors and division by zero.

---

## üìñ Syntax Guide (Keywords)

| Stylaxx Keyword | English Equivalent | Description |
| :--- | :--- | :--- |
| `dhori` | `var / int / float` | Declare a variable |
| `dekhao` | `print` | Print to console |
| `nio` | `scanf / input` | Take user input |
| `jodi` | `if` | Conditional check |
| `nahole` | `else` | Fallback condition |
| `jotokhon` | `while` | While loop |
| `ghuro` | `for` | For loop |
| `pow(a,b)` | `power` | Calculate power |
| `root(x)` | `sqrt` | Calculate square root |

## üõ†Ô∏è Installation & Build

### Prerequisites
You need the following tools installed on your system:
1.  **GCC Compiler** (MinGW for Windows)
2.  **Flex** (WinFlexBison for Windows)
3.  **Bison** (WinFlexBison for Windows)

### How to Run (Windows)
I have included a `build.bat` script to automate the process.

1.  Clone this repository or download the files.
2.  Open the folder.
3.  Double-click on **`build.bat`**.
4.  It will compile the compiler and execute the `code.laxx` file automatically.

### Manual Compilation (Linux/Mac/Windows Terminal)
If you want to compile manually, run these commands in order:

```bash
bison -d stylaxx.y
flex stylaxx.l
gcc lex.yy.c stylaxx.tab.c -o stylaxx.exe
./stylaxx.exe code.laxx
````

## üíª Example Code (`code.laxx`)

Here is a sample program written in **Stylaxx**:

```javascript
// Variable Declaration
dhori a = 10;
dhori b = 20;

// Logic Check
jodi (a < b) {
    dekhao("A is smaller than B");
} nahole {
    dekhao("A is bigger");
}

// Loop Example
dekhao("Counting 1 to 5:");
ghuro (dhori i = 1; i < 6; i = i + 1;) {
    dekhao(i);
}

// Math Example
dhori result = pow(2, 3);
dekhao("2 Power 3 is:");
dekhao(result);
```
